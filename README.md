# Despliegue de Moodle con Docker en Mac Pro 2019

Este repositorio contiene la configuración para desplegar Moodle usando Docker Compose.

## Instrucciones de uso

1.  Clonar el repositorio:
    bash
    git clone [https://github.com/Zombieboy97/moodle-compose-2.git](https://github.com/Zombieboy97/moodle-compose-2.git)
    cd moodle-compose-2
    

2.  Levantar el servicio:
    bash
    docker compose up -d
    

3.  Acceder:
    * URL: http://localhost:8080
    * Usuario: admin
    * Contraseña: Admin123!

## Evidencia
El sistema se probó en un entorno con Docker instalado y funcionando correctamente.
